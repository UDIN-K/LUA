Udink Mod v3.3.9 - Diep.io Enhancement Suite

The ultimate Diep.io mod with powerful features to enhance your gameplay!

Combat Features

  • Aimbot - Automatically aims at nearest enemy (Toggle: B) - Now aims directly at target!
  • Triggerbot - Auto-fires when enemy in crosshair
  • Prediction Aim - Leads shots based on enemy movement
  • Smooth Aim - Gradual aim interpolation for natural movement
  • Auto Fire - Continuous automatic shooting (Toggle: F)
  • Auto Spin - Automatic tank rotation (Toggle: R)
  • Threat Priority - Prioritize nearby threats to stay alive!

Visual Features

  • ESP - See enemy boxes through obstacles (Toggle: N)
  • ESP Lines - Draw lines to enemy players (prioritizes players over farm)
  • Custom Crosshair - Enhanced aiming reticle
  • Threat Indicator - Warning when enemies approach
  • Zoom Hack - Extended field of view (Toggle: Z)
  • FPS Counter - Real-time performance display
  • Coordinates Display - Show your position on map

Build Upgrader (NEW!)

  • Custom Builds - Create & save your favorite stat builds
  • Quick Spawn - One-click spawn with saved build
  • Visual Editor - Easy point allocation with +/- buttons
  • Default Builds - Glass Cannon, Rammer, Bullet, Hybrid
  • Username Save - Remember spawn names

Misc Features

  • Auto Respawn - Automatically respawn after death
  • Draggable Menu - Menu icon can be dragged anywhere
  • Customizable Colors - Change ESP and UI colors
  • Customizable Keybinds - Set your own hotkeys
  • Anti-Anti-Cheat - Bypass detection systems

Developer Options (NEW!)

  • Show Collisions - See hitboxes of all entities (debug)
  • Native FPS - Game's built-in FPS counter
  • Raw Health Values - See HP as numbers
  • Hide Game UI - Hide game interface for clean view

Modern UI

  • Glassmorphism Design - Sleek dark gradient theme
  • Smooth Animations - Beautiful transitions
  • iOS-style Controls - Toggle switches, badges
  • 7 Tab Navigation - Combat, Visuals, Builds, Misc, Colors, Keys, About

Recent Fixes (v3.3.2-3.3.9)

  • Aimbot - Now aims directly at target position (not fixed distance)
  • Farm Mode - Properly targets farm shapes with ESP lines
  • Player Detection - Multi-criteria system (type + color + radius)
  • ESP Lines - Now correctly targets enemy players, not farm blocks
  • All Mode - New priority mode that targets everything with player priority
  • Zoom Hack - Slider and toggle now work correctly
  • Self-targeting - Fixed issue where aimbot would target yourself
  • Team Mode Support - Won't target players on same team
  • Developer Tools - Debug options for collision, FPS, health values

Default Keybinds

  • V - Open/Close Menu
  • B - Toggle Aimbot
  • N - Toggle ESP
  • F - Toggle Auto Fire
  • R - Toggle Auto Spin
  • Z - Toggle Zoom Hack

Priority Modes

  • All (Player Priority) - Target everything, but prioritize players first!
  • Threat First - Prioritize nearby threats (Player > Drone > Crasher)
  • Players Only - Only target enemy players
  • Farm Only - Only target shapes (Pentagon, Triangle, Square)
  • Nearest - Target whatever is closest

Installation

  1. Install Tampermonkey or Greasemonkey extension
  2. Install this script
  3. Go to diep.io
  4. Press V to open the mod menu
  5. Drag the toggle button anywhere you like!
  6. Use Builds tab to create custom stat builds
  7. Enjoy!
